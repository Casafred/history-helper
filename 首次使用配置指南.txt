PatentLens 是一款专利审查文档获取与梳理工具，首次使用前需要完成以下配置。
打开应用后，点击右上角 ⚙️ 图标进入设置页面，按以下步骤配置。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第一步：AI 服务配置（必配）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI 服务是核心功能，用于审查意见梳理、文档分析、AI 对话等。
至少需要配置一个 AI 服务商。

【设置页面 → AI 服务 Tab】

1. 选择 AI 服务商（三选一）：

   ┌─────────────────────────────────────────────────────────┐
   │ 智谱 AI (GLM)                                          │
   │   推荐模型：glm-4-plus                                  │
   │   API 地址：https://open.bigmodel.cn/api/paas           │
   │   申请地址：https://open.bigmodel.cn                    │
   │   说明：国内服务，注册即送额度，推荐首次使用              │
   ├─────────────────────────────────────────────────────────┤
   │ DeepSeek                                               │
   │   推荐模型：deepseek-chat                               │
   │   API 地址：https://api.deepseek.com                    │
   │   申请地址：https://platform.deepseek.com               │
   │   说明：性价比高，中文能力强                             │
   ├─────────────────────────────────────────────────────────┤
   │ OpenAI 兼容                                             │
   │   推荐模型：gpt-4o                                      │
   │   API 地址：https://api.openai.com（可自定义）           │
   │   说明：支持任何 OpenAI 兼容 API（如中转站、本地部署）    │
   └─────────────────────────────────────────────────────────┘

2. 填入 API Key：
   - 在对应服务商官网注册账号 → 创建 API Key → 粘贴到输入框
   - API Key 仅保存在本地浏览器中，不会上传到任何服务器

3. 点击「测试连接」验证配置是否正确
   - 显示"连接成功"即可

4. 点击「保存」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第二步：OCR 配置（可选）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OCR 用于 PDF 文档文字提取，是翻译功能的前置步骤。

【设置页面 → OCR Tab】

1. 选择 OCR 引擎：

   ┌─────────────────────────────────────────────────────────┐
   │ PaddleOCR-VL（默认）                                    │
   │   免费，无需 API Key                                    │
   │   识别精度：中等                                        │
   │   适合：日常使用，无需额外配置                           │
   ├─────────────────────────────────────────────────────────┤
   │ GLM OCR                                                │
   │   需要智谱 API Key（如已配置智谱 AI 服务可自动复用）     │
   │   识别精度：较高                                        │
   │   适合：对识别精度有要求的场景                           │
   └─────────────────────────────────────────────────────────┘

2. 如选择 GLM OCR：
   - 填入智谱 API Key（留空则自动复用 AI 服务中智谱的 Key）
   - 点击「保存」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第三步：翻译配置（可选）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

翻译功能可使用独立的模型，与 AI 分析使用不同服务商。

【设置页面 → 翻译 Tab】

1. 翻译模型服务商：
   - 选择「跟随 AI 服务设置」→ 使用与 AI 分析相同的服务商和 Key
   - 或单独选择一个服务商 → 可使用不同的 Key 和模型

2. 翻译模型（各服务商默认翻译模型）：

   ┌─────────────────────────────────────────────────────────┐
   │ 智谱 AI：glm-4-flash（快速，免费额度内）                │
   │ DeepSeek：deepseek-v4-flash（快速，低成本）              │
   │ OpenAI：gpt-4o-mini（快速，低成本）                      │
   └─────────────────────────────────────────────────────────┘

3. 默认翻译目标语言：中文（推荐）

4. 点击「保存」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第四步：提示词配置（可选，高级）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

可自定义 AI 分析使用的提示词，调整分析角度和报告格式。

【设置页面 → 提示词 Tab】

提供 5 个可自定义的提示词：
  - 审查意见梳理提示词（含溯源版）— 有 OCR 提取结果时使用
  - 审查意见梳理提示词（简化版）— 无 OCR 结果时使用
  - 单文档分析提示词 — 阅读器中单文档分析
  - 审查历史梳理提示词 — 审查历史概要分析
  - 引用文献梳理提示词 — 引用文献分析

每个提示词均可点击「恢复默认」还原为内置版本。
建议：首次使用保持默认，熟悉后再按需调整。

快速配置方案（推荐）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果你只想快速开始，按以下最简配置即可：

  方案 A（零成本）：
    AI 服务 → 智谱 AI → 填入 API Key → 测试 → 保存
    OCR    → PaddleOCR-VL（默认，无需改动）
    翻译   → 跟随 AI 服务设置（默认，无需改动）

  方案 B（高性价比）：
    AI 服务 → DeepSeek → 填入 API Key → 测试 → 保存
    OCR    → PaddleOCR-VL（默认）
    翻译   → 跟随 AI 服务设置（默认）


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
API Key 申请指引
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【智谱 AI】
  1. 访问 https://open.bigmodel.cn
  2. 注册账号并登录
  3. 进入「API 密钥」页面
  4. 点击「创建密钥」
  5. 复制生成的 API Key

【DeepSeek】
  1. 访问 https://platform.deepseek.com
  2. 注册账号并登录
  3. 进入「API Keys」页面
  4. 点击「创建 API Key」
  5. 复制生成的 API Key

【OpenAI】
  1. 访问 https://platform.openai.com
  2. 注册账号并登录
  3. 进入「API Keys」页面
  4. 点击「Create new secret key」
  5. 复制生成的 API Key
  注：如使用中转站，Base URL 填入中转站地址即可

常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 测试连接失败怎么办？
A: 1) 检查 API Key 是否正确  2) 检查 Base URL 是否正确
   3) 检查网络是否能访问对应 API  4) 检查账号余额是否充足

Q: 可以同时配置多个 AI 服务商吗？
A: 可以。在 AI 服务 Tab 中切换服务商并分别填入 Key 保存，
   切换时选择需要的服务商即可。

Q: 翻译和 AI 分析可以用不同的服务商吗？
A: 可以。在翻译 Tab 中单独选择服务商和模型即可。

Q: OCR 提取失败怎么办？
A: 1) 确认 PDF 文档已成功加载  2) 尝试切换 OCR 引擎
   3) 如使用 GLM OCR，确认 API Key 有效

Q: API Key 安全吗？
A: API Key 仅保存在本地浏览器 localStorage 中，不会上传到
   任何外部服务器。但请注意不要在公共电脑上保存 Key。

  1. 在搜索栏输入专利号（如 US12030161B2）
  2. 查看审查历史文档
  3. 点击「AI 梳理」生成分析报告
  4. 在阅读器中查看 PDF、OCR 提取、翻译、AI 对话
