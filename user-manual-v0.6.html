1 产品概述

1.1 产品简介

PatentLens 是一款面向专利从业人员的专业工具软件，支持从 USPTO/EPO/JPO/DPMA/CNIPA 等全球主要专利局自动获取审查历史文档，并通过看板式管理、AI 智能分析、全文翻译、OCR 提取等能力，帮助用户高效梳理审查意见、答复策略、引用文献及专利原文。用户只需输入专利号，软件即可自动完成文档获取、分类归档、智能分析全流程，大幅提升专利审查应对、侵权风险评估和专利调研的效率与质量。

1.2 核心功能一览

双模式专利搜索 NEW
支持"审查文档"和"专利原文"两种搜索模式：审查文档模式获取完整审查历史档案；专利原文模式直接查询 Google Patents 获取说明书、权利要求、引用文献等专利全文信息。
批量查询 NEW
支持一次输入最多 10 个专利号进行批量查询，快速获取多篇专利的原文概览信息，适合批量专利调研场景。
历史记录管理 NEW
左侧侧边栏自动保存查询历史，支持快速回看、多选删除，无需重复输入专利号即可访问之前查询过的专利。
智能审查看板
六列看板式文档管理，将审查历史文档自动分类为审查意见、答复、请求、授权、通知和其他，支持关键词与类型组合筛选。
AI 审查意见梳理
集成 DeepSeek、智谱 GLM、OpenAI 兼容等多种 AI 服务，自动梳理审查意见、答复策略、权利要求修改、引用文献分析，支持流式响应、来源标注、手动选择文档范围。
AI 问一问 NEW
在专利原文详情页、右侧悬浮弹窗、阅读器右侧面板均可使用 AI 问一问功能，基于当前专利内容进行连续对话提问，深入了解专利技术细节和法律状态。
OCR 双引擎 NEW
支持 PaddleOCR-VL（免费本地）和 GLM OCR（云端高精度）双引擎切换，自动识别扫描版 PDF 文字，清理 OCR 噪音符号。
全文翻译 NEW
支持独立翻译模型配置，全文合并翻译不分块，流式实时显示翻译结果，支持选区翻译和网页一键翻译。
溯源对照阅读器与 PDF 标注
全屏阅读器支持文本/PDF 双模式对照阅读，左侧原文高亮引用，右侧 AI 分析/翻译/对话；PDF 模式支持高亮、划线、箭头、注释四种标注工具，可导出标注后的 PDF。
专利原文查询与多窗口
点击任意专利号即可在应用内查看 Google Patents/Espacenet 详情，支持右侧悬浮弹窗同时打开多份专利标签页快速切换，引用文献可一键复制全部公开号。
浏览器扩展 NEW
提供 Chrome 浏览器扩展，在 J-PlatPat（日本专利局）和 DPMA（德国专利局）网站浏览专利时，一键跳转至 PatentLens 进行深度分析。
Word / PDF 导出
AI 分析结果可导出为格式完整的 .docx 文件（表格可编辑）；多份审查文档可合并为带封面书目信息的 PDF；PDF 标注可导出保存。

2 安装与启动

2.1 安装版安装

下载安装程序 — 获取 PatentLens_x.x.x_x64-setup.exe 文件
运行安装程序 — 双击 .exe 文件，按照安装向导提示完成安装，可自定义安装目录
启动软件 — 从桌面快捷方式或开始菜单启动 PatentLens

2.2 便携版使用

解压文件 — 将压缩包解压到任意目录（如桌面、D 盘等）
运行程序 — 双击 PatentLens.exe 即可运行，无需安装
卸载 — 直接删除整个文件夹即可完成卸载
注意
便携版中 PatentLens.exe 与相关动态库、资源文件夹必须位于同一目录下，请勿分开存放或移动主程序位置。

2.3 首次运行注意事项

首次运行时，Windows 可能弹出"Windows 已保护你的电脑"安全提示，请按以下步骤操作：

点击弹窗中的 "更多信息"
点击 "仍要运行" 按钮即可正常启动

2.4 浏览器扩展 NEW

PatentLens 提供 Chrome 浏览器扩展，支持在专利局官网一键跳转分析：

  • 支持网站：J-PlatPat（日本专利局）、DPMAregister（德国专利商标局）
  • 功能：在专利详情页自动显示"在 PatentLens 中打开"按钮，点击即可在本应用中打开该专利进行深度分析
  • 安装方式：
    1. 打开 Chrome 浏览器，进入 chrome://extensions/
    2. 开启右上角"开发者模式"
    3. 点击"加载已解压的扩展程序"
    4. 选择软件安装目录下的 browser-extension/ 文件夹

3 界面布局

3.1 整体布局结构

PatentLens 采用侧边栏 + 顶部搜索栏 + 主内容区的现代化布局，支持亮色/暗色主题切换：

  • 左侧：历史记录侧边栏（可收起/展开），快速访问查询过的专利
  • 顶部：搜索栏区域，包含搜索模式切换、专利号输入框、查询按钮、主题切换、设置按钮
  • 主区域：根据当前模式显示搜索首页、批量查询结果、专利原文详情、审查文档分析结果等内容

3.2 搜索栏与搜索模式 NEW

搜索栏位于窗口顶部，是应用的核心入口，新增双模式切换：

元素说明
搜索模式切换"审查文档"模式（默认）：获取 Global Dossier 审查历史档案；"专利原文"模式：直接查询 Google Patents 获取专利全文
专利号输入框输入专利号或申请号，支持 US、EP、JP、DE、CN、WO 等格式，系统自动识别类型
查询按钮点击开始获取数据，过程中显示加载进度
批量查询按钮在"专利原文"模式下显示，点击展开批量输入面板（最多 10 篇）
主题切换太阳/月亮图标，切换浅色/深色主题
AI 设置按钮齿轮图标，打开完整设置对话框（AI、OCR、翻译、提示词、EPO OPS）

3.3 历史记录侧边栏 NEW

左侧侧边栏自动保存您查询过的专利记录：

  • 记录列表：按时间倒序显示历史查询的专利号和标题
  • 快速访问：点击记录项即可重新加载该专利数据，无需重复输入
  • 多选删除：点击勾选图标进入多选模式，可批量选中并删除历史记录
  • 清空全部：一键清空所有历史记录
  • 收起/展开：点击边缘箭头可收起侧边栏，获得更大阅读空间

3.4 标签页区域

在"审查文档"模式下加载专利后，主区域显示 4 个功能标签页：

  • 概览 — 专利基本信息 + 审查时间线
  • 同族 — IP5 五局同族专利信息
  • 审查看板 — 六列文档看板 + 合并导出
  • AI 分析 — AI 审查意见梳理、引用文献分析

3.5 概览标签页

概览标签页显示专利的基本信息和审查时间线：

  • 申请信息卡片 — 专利号、标题、申请人、申请日、发明人等
  • 当前状态卡片 — 专利当前审查进度状态
  • 审查时间线 — 可视化时间轴展示审查历程关键事件，彩色标记区分类型（审查意见红、答复蓝、请求橙、授权绿、通知紫）

3.6 同族标签页

同族标签页展示 IP5 五局同族专利信息：

  • 同族成员列表 — 同一专利族在 USPTO/EPO/JPO/KIPO/CNIPA 的申请情况
  • 各成员状态 — 显示每个同族成员的审查状态和公开信息

3.7 审查看板标签页

审查看板是文档管理核心区域，顶部提供"合并导出"按钮，六列看板自动分类审查文档：

  • 筛选栏 — 关键词搜索 + 类型标签组合筛选
  • 六列看板 — 审查意见、答复、请求、授权、通知、其他
  • 文档卡片 — 显示文档代码、名称、日期、类型标签，点击展开文本
  • 卡片操作 — 手动提取文本、下载 PDF 原文

3.8 AI 分析标签页

AI 分析标签页提供智能梳理功能：

  • 手动选择面板 — 默认展开，支持搜索过滤、全选/全不选/默认选择，已选状态自动记忆
  • 梳理审查意见 — AI 综合分析选中的审查文档，生成结构化审查历史分析报告
  • 梳理引用文献 — AI 专门分析审查意见中引用的对比文献
  • 分析结果展示 — Markdown 格式渲染，专利号自动识别为可点击链接
  • 操作按钮 — 阅读器（溯源对照）、导出 Word、中止分析

4 快速入门

本章以完整操作流程，带您快速上手 PatentLens 的核心功能。

4.1 配置 AI 服务

使用 AI 相关功能前，需先配置 AI 服务的 API Key：

点击搜索栏右侧的 齿轮图标，打开"设置"对话框
在 "AI 服务" 标签页选择服务商（DeepSeek / 智谱 GLM / OpenAI 兼容）
输入对应服务商的 API Key
（可选）修改 API 端点 地址以适配代理或私有部署，选择要使用的模型
（可选）在 "OCR" 标签页选择 OCR 引擎（PaddleOCR-VL 免费本地 / GLM OCR 云端高精度）
（可选）在 "翻译" 标签页独立配置翻译模型
点击 "测试连接" 验证配置正确后关闭对话框
提示
API 端点地址默认已填充各服务商标准地址，一般无需修改。翻译模型可独立于主分析模型配置，默认使用性价比更高的模型（如 glm-4-flash、deepseek-v4-flash）。

4.2 搜索专利

选择搜索模式："审查文档"获取审查历史；"专利原文"直接查看专利全文
在 专利号输入框 中输入专利号（如 US12030161B2、EP4252965A3），系统自动识别类型
点击 查询按钮，等待数据加载完成

4.3 使用 AI 问一问 NEW

在专利原文详情页，您可以直接向 AI 提问了解专利详情：

切换到 "专利原文"模式查询专利，或在审查文档中点击任意专利号链接
点击顶部 "AI 问一问" 按钮，打开 AI 对话面板
在对话框中输入问题（如"这个专利的核心发明点是什么？"、"独立权利要求有哪些限制？"），AI 会基于专利全文内容回答
支持 连续多轮对话，可以基于上下文深入追问

4.4 浏览审查看板

在"审查文档"模式下加载专利后，点击 "审查看板" 标签页
文档自动归入六列看板：审查意见、答复、请求、授权、通知、其他
使用顶部 筛选栏 按关键词或类型快速定位文档
点击文档卡片 展开查看 提取的文本内容

4.5 AI 分析审查意见

切换到 "AI 分析" 标签页，手动选择面板默认展开
在手动选择面板中勾选需要分析的文档（默认已勾选审查意见和答复），可使用搜索框快速过滤
点击 "梳理审查意见" 按钮，AI 开始分析，结果流式实时显示
分析过程中可随时点击红色 "中止梳理" 按钮中断
（可选）点击 "梳理引用文献" 专门分析审查意见中的对比文献

4.6 导出与阅读

点击 "阅读器" 按钮，打开全屏溯源对照阅读器，对照查看原文与 AI 分析，支持 PDF 标注
点击 "导出 Word" 按钮，将 AI 分析结果导出为格式完整的 .docx 文件
在审查看板顶部点击 "合并导出"，将所有审查文档合并为一个带封面的 PDF 文件
完成
至此，您已掌握 PatentLens 的核心操作流程。更多高级功能请参阅第 5 章功能详解。

5 功能详解

5.1 双模式专利搜索 NEW

PatentLens 提供两种搜索模式，覆盖不同使用场景：

审查文档模式

通过 Global Dossier API 获取专利的完整审查历史档案，包括：

  • 各次审查意见通知书（Office Action）
  • 申请人答复文件（Response/Amendment）
  • 授权通知书、各类请求和官方通知
  • 自动分类归档到审查看板

专利原文模式

直接查询 Google Patents 获取专利全文信息，包括：

  • 专利说明书全文（含背景技术、发明内容、具体实施方式）
  • 权利要求书
  • 引用专利列表、被引用专利列表、相似专利
  • 法律状态信息
  • 支持一键下载 PDF 原文、Espacenet 跳转查询

支持的专利号格式

专利局代码专利号示例

欧洲专利局EPOEP4252965A3、EP12345678
日本特许厅JPOJP12345678、JP2020123456A
德国专利商标局DPMADE12345678
中国国家知识产权局CNIPACN112345678A、CN12345678B
WIPO/PCTWIPOWO2020123456A1

5.2 批量查询 NEW

在"专利原文"模式下，点击输入框右侧的批量查询按钮（四方格图标），可一次输入最多 10 个专利号批量查询：

  • 输入方式：每行输入一个专利号，系统自动识别格式
  • 进度显示：实时显示查询进度（如"3/10 已完成"）
  • 结果展示：卡片网格形式展示每篇专利的概览信息（标题、申请人、申请日等）
  • 快捷操作：点击卡片即可进入该专利的详情页进行 AI 问一问、查看引用文献等操作

5.3 历史记录管理 NEW

左侧历史记录侧边栏帮助您快速访问之前查询过的专利：

  • 自动保存：每次查询成功后自动保存到历史记录
  • 快速回看：点击记录项即可重新加载数据（有缓存时优先使用缓存）
  • 多选操作：点击顶部勾选图标进入多选模式，可勾选多项后批量删除
  • 清空历史：点击垃圾桶图标清空所有记录（此操作不可恢复）

5.4 概览

概览标签页提供专利的摘要信息和审查历程时间线：

  • 申请信息卡片：专利号、标题、申请人、申请日、发明人、当前法律状态
  • 审查时间线：按时间正序/倒序展示审查关键事件，彩色标记区分类型，点击可跳转到对应文档

5.5 同族

同族标签页展示 IP5 五局同族专利信息，帮助了解专利全球布局。点击同族专利号可直接查看该同族成员的详情。

5.6 审查看板

审查看板采用六列看板式布局自动分类管理审查文档：

列名文档类型颜色标识默认纳入 AI
审查意见驳回、限制性要求、审查意见通知书红色是
答复申请人意见陈述、修改文件蓝色是
请求RCE、审查请求、延期请求等橙色否
授权授权通知书、缴费通知绿色否
通知各类官方通知紫色否
其他说明书、权利要求、其他未分类文档灰色否

筛选与操作

  • 关键词搜索：输入框实时过滤卡片
  • 类型筛选：点击类型标签筛选
  • 卡片展开：点击展开/收起 OCR 提取的文本
  • 提取文本：手动触发 OCR 文字提取
  • 下载 PDF：下载原始 PDF 文档
  • 列独立滚动：各列可独立滚动，互不影响

5.7 AI 审查意见梳理

AI 审查意见梳理是核心分析功能，从竞争对手方视角生成结构化分析报告：

手动选择面板

  • 默认展开：进入 AI 分析标签页即可看到，方便确认文档范围
  • 搜索过滤：输入关键词快速定位文档
  • 快捷选择：全选、全不选、默认选择（审查意见+答复，排除 SPEC）
  • 已选摘要：底部显示"已选 X 篇：审查意见 X · 答复 X"实时统计
  • 选择记忆：选择状态自动保存，切换标签页后恢复

分析内容

梳理审查意见生成的报告包含：

  1. 案件概要 — 专利基本信息、当前阶段、侵权影响概述
  2. 审查对话时间线 — 按时间列出来回审查沟通，每轮对侵权方的意义
  3. 争议焦点辩驳详表 — 按争议点组织审查员论点 vs 申请人反驳，评估反驳有效性
  4. 权利要求范围演变 — 审查意见-答复策略-权利要求修改-范围变化-规避影响-禁反言风险 联动对照
  5. 关键无效机会 — 审查员引用但未完全采纳的文献、申请人修改后仍存的缺陷
  6. 授权前景与风险评级 — 授权可能性、权利要求范围宽度、规避难度评级
  7. 监控建议 — 建议关注的审查进展节点

来源标注

AI 分析报告中每一句话、每个结论都会标注原文来源，格式为【来源: D0_B_p1_0】，对应"第 X 份文档第 Y 页第 Z 块"，在阅读器中点击可直接跳转到原文对应位置，实现可溯源的智能分析。

专利号自动链接
AI 分析中引用的对比文献专利号会自动保留完整格式（如 US7456789B2），并自动转换为可点击链接，点击即可在右侧悬浮弹窗中查看该专利详情，支持多标签页快速切换。

6 常见问题

Q1：首次运行 Windows 提示"已保护你的电脑"怎么办？

这是 Windows SmartScreen 正常提示。点击"更多信息" → "仍要运行"即可。PatentLens 是安全的本地应用。

Q2：搜索专利时获取文档失败怎么办？

  • 检查专利号格式是否正确，系统是否正确识别了专利局
  • 检查网络连接是否正常
  • 确认该专利确实有公开的审查历史文档
  • 部分专利局服务器可能有访问频率限制，稍后重试

Q3：AI 分析/问一问失败怎么办？

  • 确认 API Key 正确填写
  • 确认网络可访问对应 AI 服务商
  • 检查 API 端点地址（使用代理需修改）
  • 确认所选模型可用，账户余额充足
  • 在设置中点击"测试连接"验证配置

Q4：PaddleOCR 无法使用怎么办？

  • 确认已安装 Python 3.8+
  • 安装 Python 依赖：pip install -r requirements.txt
  • 或者切换到 GLM OCR 云端引擎（需要智谱 API Key）

Q5：如何中止正在进行的 AI 分析？

AI 分析过程中界面显示红色"中止梳理"或"中止引用梳理"按钮，点击即可中断。已生成的部分结果会保留。

Q6：为什么 SPEC（说明书）默认不纳入 AI 分析？

SPEC 通常篇幅很长，纳入分析会消耗大量 Token 且对审查意见梳理帮助有限。默认排除可提高效率、降低成本。如需包含，可在手动选择面板中勾选。

Q7：导出的 Word 文件中表格可以编辑吗？

可以。导出的 .docx 中 Markdown 表格已渲染为真正的 Word 表格，支持直接编辑单元格、调整列宽、修改样式。

Q8：PDF 标注会保存吗？

标注数据临时保存在当前会话中。请在关闭阅读器前点击"导出标注后文档"按钮，将标注叠加到 PDF 上永久保存。

Q9：右侧悬浮弹窗支持同时打开多少个专利？

支持在弹窗内打开多个标签页快速切换，关闭弹窗时所有标签页一起关闭。如需同时对比多份专利，建议使用全屏详情页分别查看。

Q10：历史记录保存在哪里？会上传吗？

所有配置（API Key、历史记录、自定义提示词等）都保存在本地浏览器存储中，不会上传到任何服务器。
